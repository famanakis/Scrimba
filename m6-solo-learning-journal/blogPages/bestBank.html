BestBank CSS to the nth

What the "nth" is that?! This project was one that grew my CSS skills for working with elements to the nth degree.

In CSS, the :nth-child() and :nth-of-type() selectors can be used to select specific elements based on their position within their parent container. These selectors use an index number to select the element based on its position, starting from 1.

Here are some examples of how to use these selectors:

                /* Selects the second element of div siblings */
                div:nth-child(2) {
                  background: red;
                }
            
                /* Selects the second li element in a list */
                li:nth-child(2) {
                  background: lightgreen;
                }

Note that these selectors can be used with other selectors and combinators to create more complex selection rules.

In the BestBank project, the Spendings column is rendered using :nth Child() selectors to vary the width of each 'li' in the Spendings list.

                .spendings {
                    display: flex;
                    justify-content: space-between;
                    align-items: center;
                    height: 42px;
                    border-radius: 15px;
                    padding: 0 20px;
                    background-color: var(--orange);
                }

                .spendings-list {
                    width: 453px;
                    gap: 12px;
                }

                .spendings-list li:nth-child(2) {
                    width: 419px;
                }

                .spendings-list li:nth-child(3) {
                    width: 329px;
                }

                .spendings-list li:nth-child(4) {
                    width: 255px;
                }

                .spendings-list li:nth-child(5) {
                    width: 217px;
                }
