Secure Password Generator

This project taught me how to generate random sequences of numbers, letters, and characters by targeting user input to create unique passwords. I had a lot of fun finding ways to customize this project, to include changing the password length, choosing which characters to include in the passwords, and having a light/dark mode toggle theme for the app. Here is how I chose to create my own random password app.

To begin this project, I needed to define a sets of letters, numbers and special characters. I did this by creating const variables for lowercase letters, capital letters, numbers, and special characters. I created four different const variables because I wanted the ability to use or not use certain types of characters in my passwords.

Then, I used a function to generate a custom array based off of the selected parameters. Sometimes the array would hold letters and numbers, while other times only letters. Then a random password would be generated from the custom array. The random password generation was accomplished through a random password function that used sort, slice, and Math.random methods.

This is the code I used to generate a random password

                function randomPassword(arr,num) {
                    const shuffled = arr.sort(() => 0.5 - Math.random())
                    return shuffled.slice(0,num)   
                }

This is the code I used to generate a random passwords for my app:

                function generatePassword() {
                    const passwordLength = sliderValue.value
                    nums.checked && chars.checked ? customArr = (lowerCase.concat(upperCase, numbers, symbols)) :
                    nums.checked && !chars.checked ? customArr = (lowerCase.concat(upperCase, numbers)) :
                    chars.checked && !nums.checked ? customArr = (lowerCase.concat(upperCase, symbols)) :
                    customArr = lowerCase.concat(upperCase)
                    const newPassword = randomPassword(customArr.split(''),passwordLength).join("")
                    iconVisibility()
                    return newPassword 
                }
